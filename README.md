# Sports Center Management

A full-stack web application for managing sports facility reservations. The application provides a complete solution for customers to view available facilities and create bookings, while administrators can manage facilities and reservation data.

The project focuses on database design, CRUD operations, user interaction, and responsive web development.

## Main Features
- Facility listing and availability checking
- Online reservation management
- Database-driven application
- Booking conflict prevention
- CRUD operations for customers, facilities, and reservations
- Responsive user interface

## Tech Stack
**Frontend:** HTML, CSS, JavaScript  
**Backend:** PHP  
**Database:** MySQL  
**Tools:** XAMPP, phpMyAdmin, Git, VS Code

## Database Structure
The application uses a relational database containing:
- Customers
- Sports facilities
- Reservations

## Purpose
Developed as a school project to demonstrate practical knowledge of web development, backend programming, and relational database management.
